# Итоговый проект: ETL-скрипт для данных грейдера

Скрипт загружает данные об использовании грейдера студентами из API,
обрабатывает, валидирует и сохраняет в локальную PostgreSQL.

## Быстрый старт

```bash
# Поднять PostgreSQL
docker compose up -d

# Данные за вчера
python main.py

# Данные за период
python main.py 2024-01-01 2024-01-15
```

Перед запуском — скопируйте `.env.example` в `.env` и заполните своими значениями:

```bash
cp .env.example .env
```
